# TodoApp

Flutter Todo application

![Alt text](git-img/flutterfire.png?raw=true "FlutterFire")  

## Features

* Login with **Google** and **Firebase**  
* Add thing todo  
* Check realised task  
* Remove task  

## Requirements

Refresh Gradle

```sh
cd .\todo_app\android\  
./gradlew clean build
./gradlew --refresh-dependencies  
```
