# todo-app

Flutter Todo application

![Alt text](assets/flutterfirebase.png?raw=true "FlutterFire")  

## Features

* Login with **Google**  
* Add task in **Firebase**    
* See task details  
* Check realised task  
* Remove task  

## Requirements

Refresh Gradle

```sh
cd .\todo_app\android\  
./gradlew clean build
./gradlew --refresh-dependencies  
```
